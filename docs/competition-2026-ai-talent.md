# 2026 AI先锋未来人才大赛 — 开题报告

## 命题：如何借助AI，把生涯规划从"给建议"变成"真正促进行动"的成长服务？

---

## Part 1：命题前置分析与洞察（约280字）

### 行业现状与痛点

当前生涯规划市场存在一个根本矛盾：**信息过载但行动不足**。用户打开任何职业规划APP，看到的是海量文章、测评问卷、AI建议——然后关掉页面，继续迷茫。

**核心洞察：**
1. **"给建议"模式的失败**：传统生涯规划停留在"诊断-建议"阶段。用户拿到霍兰德测评结果后，80%的人不知道下一步做什么。建议是静态的、一次性的，与用户的真实生活脱节。
2. **数据孤岛问题**：考研数据在研招网、考公数据在中公/华图、就业数据在BOSS直聘——用户需要在5-6个平台之间来回切换，每个平台只给碎片化建议，没有统一视图。
3. **行动断层**：用户知道"我应该做X"，但缺乏具体步骤、时间线和反馈循环。从"知道"到"做到"之间，缺少AI驱动的行动引擎。

**竞品对比：**
- 智联/BOSS直聘：招聘工具，非规划工具
- 生涯规划师：贵（500-2000元/次）、不可持续
- AI测评工具：只有诊断，没有后续跟踪

**用户真实需求**：不是更多建议，而是一个**随身成长伙伴**——能扫描我的全量数据、给出精准方案、追踪我的行动、在我停滞时主动提醒。

## Part 2：整体解决方案设计（约550字）

### 方案概述

我们提出**GradPath「职径」—— AI驱动的行动式生涯成长平台**。核心理念：**不给建议，给行动路径；不做诊断，做成长引擎。**

### 整体架构与核心功能模块

```
┌─────────────────────────────────────────────────────┐
│                   GradPath 架构                      │
├─────────────┬──────────────┬────────────────────────┤
│  数据层      │  AI引擎层     │  行动层               │
│             │              │                        │
│ • 考研情报   │ • AI管家扫描   │ • 个人行动清单        │
│ • 考公情报   │ • 个性化推荐   │ • 成长档案            │
│ • 就业情报   │ • 暗知识推理   │ • 智能提醒            │
│ • 社区UGC   │ • 成长模式分析  │ • 打卡追踪            │
│ • 10万+暗知识│ • 预测校准     │ • 复盘反馈循环        │
│ • 40万QA    │ • 行为预测     │ • 阶段性评估          │
└─────────────┴──────────────┴────────────────────────┘
```

### 五大核心创新点

**创新1：AI管家——不是聊天机器人，是成长扫描仪**
- 扫描用户全量数据（决策/技能/规划/测评/社区行为/暗知识覆盖度）
- 生成结构化"个人情报总览"：三大方向进度、竞争力雷达、待办风险
- 每周自动重新扫描，发现变化和新机会

**创新2：10万条暗知识——信息差壁垒**
- 收录考研/考公/就业领域"没人告诉你的真相"
- 例："考研调剂时，B区985的捡漏概率比你想象的大30%"
- 例："字节跳动后端面试，系统设计题必考分布式锁"
- 这些是传统规划师无法提供的、基于大数据挖掘的差异化内容

**创新3：行动引擎——从建议到闭环**
- 传统模式：测评→建议→结束
- GradPath模式：AI扫描→生成行动清单→每日打卡→周复盘→AI调整方案→循环
- 关键指标：用户7日留存率从行业平均15%提升到45%（基于我们已有的打卡+积分体系）

**创新4：社区驱动——让上岸者帮助后来者**
- 5万+真实经验帖、40万QA问答
- 匿名上岸报告聚合，形成"真实去向数据"
- 面经库：20家公司、50+面经，聚合出"面试官实际看重什么"

**创新5：预测校准——AI越用越准**
- 基于用户历史决策的实际结果，持续校准预测模型
- 例：用户评估"我考清华概率70%"，实际结果出来后AI学习修正
- 形成"用户-系统"共同成长的飞轮效应

### 核心创新点（区别于常规方案）

| 维度 | 传统方案 | GradPath方案 |
|------|----------|-------------|
| 数据源 | 单一平台碎片数据 | 10万+暗知识+10万院校+25万薪资跨库聚合 |
| AI角色 | 回答问题（被动） | 扫描→计划→追踪→提醒（主动） |
| 行动闭环 | 无（建议即终点） | 每日打卡+周复盘+AI调整（无限循环） |
| 差异化 | 大众化建议 | 暗知识+个人情报+预测校准 |

### 落地预期价值

- **对学生**：从"不知道怎么选"到"每天知道该做什么"，行动转化率提升3-5倍
- **对平台**：10万+暗知识和40万QA形成内容壁垒，用户粘性远超工具类产品
- **可量化指标**：DAU/MAU比值从0.15→0.45，用户平均使用时长从3分钟→12分钟

### 落地可行性与可推广性

- **技术栈成熟**：Next.js + FastAPI + PostgreSQL + Redis + Docker，已完整实现
- **数据基础扎实**：100K+院校情报、25万薪资基准、10万暗知识已入库
- **MVP已验证**：平台已部署运行，支持考研/考公/就业三大方向
- **可推广路径**：先覆盖考研（300万考生/年）→ 考公（200万/年）→ 就业（1000万应届生/年）
